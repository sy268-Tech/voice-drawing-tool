# AI 语音绘图工具 🎨

基于 **Vite + React + TypeScript + SVG** 的语音/文字绘图应用。支持浏览器 Web Speech API 中文语音识别，以及**大模型（LLM）智能解析**和**本地规则解析**双引擎。

## ✨ 功能亮点

- 🎤 **语音识别** — 浏览器 Web Speech API，中文语音转文字
- 🧠 **LLM 智能解析** — 接入 OpenAI 兼容大模型，理解任意自然语言绘图指令
- 📐 **本地规则引擎** — 无需网络即可处理丰富指令，LLM 失败自动回退
- 🎨 **SVG 对象绘图** — 每个图形独立可操作
- 🔄 **序号定位** — 支持「第一个圆」「最后一个矩形」
- 📏 **可控线条** — 指定像素长度，以线为半径画圆
- 🎯 **参考点操作** — 从点延伸、以点为圆心

---

## 🚀 快速开始

```bash
npm install
npm run dev
```

用 **Chrome** 打开终端显示的地址（默认 http://localhost:5173），点击「开始语音」或直接在输入框打字。

> ⚠️ Web Speech API 依赖 Google 服务器，国内网络可能不可用。界面会自动切换到文字输入模式。

## 🗣️ 指令参考

### 绘图（draw）

| 指令 | 效果 |
|------|------|
| `画一个红色圆形` | 在中央画红色圆 |
| `在右上角画一个黄色太阳` | 右上角画太阳 |
| `画蓝色天空和绿色草地` | 批量绘制背景（逗号/和/然后拆分） |
| `画一棵树` | 中央画树 |
| `在左上角写 Hello` | 左上角写文字 |
| `画一个点` | 画一个小圆点 |
| `在中间画一个红色的点` | 指定位置和颜色的点 |

### 线条 & 长度（line / extend）

| 指令 | 效果 |
|------|------|
| `画一条长度200的线` | 200px 水平线 |
| `画一条100像素的线` | 100px 线 |
| `画200px的线` | 同上 |
| `从这个点向右延伸一条线` | 从上一个图形向右画 150px 线 |
| `向右延伸200` | 向右 200px |
| `向上延伸100像素` | 向上 100px |
| `画一条长度200的线，然后以这条线为半径画圆` | 线长 = 圆半径 |

支持 8 个方向：左/右/上/下/左上/右上/左下/右下

### 以线为半径画圆

| 指令 | 效果 |
|------|------|
| `以这条线为半径画圆` | 圆心在线起点，半径 = 线长 |
| `以这条线为半径画一个红色圆` | 同上 + 指定颜色 |

### 以点为圆心

| 指令 | 效果 |
|------|------|
| `以这个点为圆心画一个圆` | 圆心在点位置 |
| `以这个点为中心画一个红色圆形` | 同上 + 颜色 |

### 移动（move）

| 指令 | 效果 |
|------|------|
| `把太阳移动到左上角` | 移动太阳 |
| `把圆移到右边` | 移动最后一个圆 |
| `将第一个矩形移动到右下角` | 移动第一个矩形 |

### 改色（recolor）

| 指令 | 效果 |
|------|------|
| `将蓝色的圆改为黄色` | 改最后一个蓝色圆的颜色 |
| `把太阳改成红色` | 改太阳颜色 |
| `把第二个圆的颜色改为绿色` | 指定序号的改色 |

### 缩放（resize）

| 指令 | 效果 |
|------|------|
| `把刚才的圆变大` | 放大 1.3 倍 |
| `把太阳变小` | 缩小 0.75 倍 |
| `把第一个矩形放大` | 放大第一个矩形 |

### 删除 / 撤销 / 清空

| 指令 | 效果 |
|------|------|
| `删掉第一个圆` | 删除最早画的圆 |
| `删除蓝色的圆` | 按颜色删除 |
| `删除刚才的太阳` | 删除最后一个太阳 |
| `撤销` | 撤销上一步 |
| `清空画布` | 清除全部 |

## 📐 复杂指令拆解

系统自动按 `，` `。` `然后` `再` `和` 拆解复合指令：

```
画蓝色天空和绿色草地，然后在右上角画一个太阳
→ 画蓝色天空
→ 画绿色草地
→ 在右上角画一个太阳
```

> 注意：拆分产生的无动词片段（如 `蓝色的圆`）会被自动忽略，不会误画图形。

## 🔧 技术栈

- **框架**：React 18 + TypeScript
- **构建**：Vite 5
- **绘图**：SVG（对象级）
- **语音识别**：Web Speech API（SpeechRecognition）
- **语音反馈**：Web Speech API（SpeechSynthesis）
- **LLM**：OpenAI 兼容 Chat Completions API
- **存储**：localStorage（API Key 持久化）
