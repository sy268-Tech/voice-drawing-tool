# 🎨 语音绘图工具 · Voice Draw

> 中英双语语音指令驱动 SVG 画布绘图 — 纯本地解析，零 AI 依赖

**说出形状、颜色、位置，即可自动绘图。** 基于浏览器 Web Speech API，无需联网、无需 API Key，说完立刻画出。

## ✨ 核心特性

- 🎤 **纯语音驱动** — Web Speech API，说完即画，支持实时识别预览与连续聆听模式
- 🌏 **中英双语** — 中文/English 指令均可识别，一键切换识别语言与界面语言
- 🧩 **零 AI 依赖** — 纯本地关键词解析，无需任何大模型 API
- 🔍 **同音字容错** — "元"→圆、"巨型"→矩形、"五角行"→五角星 等自动纠正
- 🎨 **13 种图形** — 圆/矩形/三角/椭圆/星形/直线/文字 + 箭头/菱形/心形/弧形/六边形/云朵
- 🌈 **16 种颜色** — 新增 青、金、天蓝、青柠、靛 5 色，中英文颜色名均可
- 🖱️ **画布直接编辑** — 点击选中、拖拽移动、Delete 键删除
- ↩️ **完整撤销/重做** — 历史栈支持「撤销 / 重做 / undo / redo」
- 🔄 **旋转指令** — "旋转矩形45度" / "rotate the square 45 degrees"
- 🔢 **批量绘制** — "画三个圆" / "draw three circles" 自动错开排列
- 📤 **一键导出** — 导出 SVG / 高清 PNG
- 🔢 **序号定位** — "第一个圆"、"最后一个矩形" / "the first circle"
- 📦 **复合指令** — "画红色圆形，然后画蓝色矩形" / "draw a heart then a star"

## 🚀 快速开始

```bash
cd voice-drawing-tool
npm install
npm run dev
```

用 **Chrome 浏览器** 打开 `http://localhost:5173`，点击 **🎤 点击说话** 或直接在输入框打字。

> ⚠️ Web Speech API 依赖 Google 语音服务，国内网络可能受限。可改用文字输入模式。

## 🗂️ 项目结构

```
voice-drawing-tool/
├── index.html                     # 入口 HTML
├── package.json                   # 依赖与脚本
├── tsconfig.json                  # TypeScript 配置
├── vite.config.ts                 # Vite 构建配置
└── src/
    ├── types.ts                   # 核心类型 (Drawable, Command, ShapeType)
    ├── constants.ts               # 画布配置 + 颜色/同音字关键词组
    ├── parser.ts                  # 中文语音指令解析器（含同音字模糊匹配）
    ├── engine.ts                  # 绘图引擎 (applyCommand)
    ├── useSpeechRecognition.ts    # Web Speech API Hook
    ├── main.tsx                   # App 主组件 + SVG 渲染
    └── style.css                  # 渐变紫色主题 UI
```

### 模块职责

| 模块 | 职责 |
|------|------|
| `types.ts` | `ShapeType`（13 种图形）、`Command`、`Drawable` 类型定义 |
| `constants.ts` | 画布 900×560、颜色映射、**5 组同音关键词字典**（图形/颜色/尺寸/方位/动作） |
| `parser.ts` | `matchAny()` 模糊匹配引擎 → `parseVoiceCommand()` 返回 `Command[]` |
| `engine.ts` | `applyCommand()` 将指令应用到图形数组，管理 undo/clear/CRUD |
| `useSpeechRecognition.ts` | 封装 Web Speech API，返回 `{ listening, start, stop }` |
| `main.tsx` | SVG 渲染 13 种图形 + 交互 UI |
