# 🎨 语音绘图工具 · 演示视频脚本

> 预计时长：5-6 分钟 | 录制方式：Win+G 录屏 + 麦克风配音

---

## 准备工作

1. Chrome 浏览器打开 `http://localhost:5175`
2. 确保麦克风可用（Win+G → 音频设置，录制系统声音 + 麦克风）
3. 按下 **Win+G** → 点击录制按钮（或 Win+Alt+R）
4. 按照以下脚本操作和配音

## 录制后处理建议

1. 按 **Win+G** 停止录制，视频保存在 `C:\Users\吴孟远\Videos\Captures\`
2. 将视频文件移动到 `F:\voice-drawing-tool\voice-drawing-tool\`
3. 可用 [Shotcut](https://shotcut.org/)（免费）或剪映进行剪辑，加片头片尾
4. 配音如果现场说得不好，可以后期用剪映的"文本朗读"功能重新配音
